# Clinical Data Management Dashboard

An interactive dashboard for clinical trial data analysis.

## How to Run

1. Install R and RStudio
2. Open `app.R` in RStudio
3. Click "Run App" button

## Project Structure
- `app.R` - Main dashboard application
- `data/` - Clinical data files
- `scripts/` - Data generation scripts

## Features
- Patient demographics analysis
- Lab results visualization
- Adverse event tracking
- Interactive filtering
